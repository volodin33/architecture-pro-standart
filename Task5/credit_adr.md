### <a name="_3bfxc9a45514"></a>**Функциональные требования**

#### UC1. Подача заявки на кредит на сайте (новый клиент)
#### Действующие лица: Пользователь
#### Описание
1. Пользователь открывает страницу кредитных продуктов на сайте.
2. Система показывает предложение открыть кредит по доступной ставке.
3. Пользователь выбирает кредитный продукт.
4. Пользователю отображается форма, где необходимо указать Ф. И. О. и номер телефона.
5. Пользователь заполняет форму и отправляет заявку.
6. Система создаёт заявку на кредит со статусом «Новая».
7. (Опционально) Пользователю предлагается заполнить паспортные данные для получения предварительного решения.
8. Пользователь заполняет паспортные данные и отправляет форму.
9. Система запрашивает данные из системы кредитных историй и формирует решение об одобрении / неодобрении кредита.
10. Система сохраняет результат предварительного решения по заявке.
11. Система отправляет пользователю СМС-уведомление о необходимости прийти в отделение для идентификации.
12. Система отображает пользователю сообщение о принятии заявки.


#### UC2. Подача заявки на кредит в интернет-банке (действующий клиент)
#### Действующие лица: Пользователь (клиент интернет-банка)
#### Описание
1. Пользователь открывает раздел кредитных продуктов в интернет-банке.
2. Система показывает список доступных кредитов с актуальными условиями.
3. Системой может отображаться предодобренное предложение по кредиту.
4. Пользователь выбирает кредитный продукт (или предодобренное предложение).
5. Пользователь указывает счёт зачисления средств и необходимые данные для заявки.
6. Пользователь отправляет заявку на кредит.
7. Система валидирует введённые данные.
8. Система создаёт заявку со статусом «Черновик / Ожидает подтверждения».
9. Система инициирует отправку СМС-кода на номер пользователя.
10. Пользователь вводит СМС-код.
11. Система проверяет СМС-код.
12. При успешной проверке система переводит заявку в статус «Подтверждена клиентом» и направляет её в банк.
13. Система отображает пользователю сообщение о принятии заявки.
